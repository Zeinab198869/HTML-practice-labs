
# HTML Practice Labs

A collection of HTML practice labs completed as part of my self-directed learning journey toward a SOC Analyst role. Each lab focuses on a different HTML concept — from semantic structure to forms, tables, and media elements.

## Labs

| Lab | Description |
|---|---|
| [Book Catalog Table](./book%20catalog%20table) | A table displaying book details using `<thead>`, `<tbody>`, and `<tfoot>` |
| [Calculus Final Exam Table](./calculus%20final%20exam) | A grades table with a caption and calculated average |
| [Event Hub](./event%20hub) | A semantic event listing page using `<header>`, `<nav>`, `<section>`, and `<article>` |
| [Hotel Feedback Form](./hotel%20feedback%20form) | A multi-section form with radio buttons, checkboxes, and dropdowns |
| [Audio & Video Player](./audio%20video%20player) | An HTML audio player using the `<audio>` element |
| [Video Compilation Page](./video%20compilation%20page) | A page embedding multiple videos using `<iframe>` |
| [List of Major Web Browsers](./list%20of%20major%20web%20browsers) | A description list (`<dl>`, `<dt>`, `<dd>`) of popular browsers |
| [Mr. Whiskers' Blog](./cat%20blog%20page) | A blog page using semantic HTML: `<header>`, `<main>`, `<article>`, `<footer>` |
